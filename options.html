Código configurado (Trello):

Jira
Para configurar um domínio Atlassian (Jira), insira apenas o valor do host utilizado. Ex: inspira.atlassian.net
Trello
Atualmente, a configuração de boards Trello é feita apenas de forma genérica, sem diferenciar entre boards. Para configurar o domínio Trello, basta inserir trello.com. Como boards Trello não possuem código de issue, este padrão será configurado manualmente, logo após a inclusão do domínio. Ex: Para obter a descrição de um card como "[RDG-23] Teste", será necessário configurar o código de issue como RDG.
